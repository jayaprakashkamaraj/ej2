Keyboard Navigation Sample

Key

Usage
Alt + downArrow - datepicker popup open
Alt + downArrow + Alt + downArrow - toggle popup
DownArrow - navigate to next down element
UpArrow - navigate to next up element
RightArrow - navigate to next right element in the datepicker
LeftArrow - navigate to next left element in the datepicker
Ctrl + upArrow - navigate to top view in the datepicker
Ctrl + downArrow - navigate to lower view in the datepicker
Esc - close the popup
